# VacationVibe - פלטפורמת הזמנת חופשות (גרסה מלאה)

פלטפורמת הזמנת חופשות Full-Stack עם React, NestJS, TypeORM, Redux Toolkit ו-Material-UI.

## תכונות

- גלישה וחיפוש יעדי חופשה
- מערכת הזמנות מלאה
- ממשק ניהול למנהלים
- אימות משתמשים עם JWT
- עיצוב רספונסיבי עם Material-UI
- API מבוסס NestJS

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| React | ספריית UI |
| NestJS | פריימוורק Backend |
| TypeORM | ORM לבסיס נתונים |
| Redux Toolkit | ניהול מצב |
| Material-UI | עיצוב ממשק |
