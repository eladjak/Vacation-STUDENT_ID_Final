### 2. בדיקות
```bash
# בדיקות יחידה
npm run test:unit

# בדיקות אינטגרציה
npm run test:integration

# בדיקות E2E
npm run test:e2e

# בדיקת כיסוי
npm run test:coverage

# בדיקות אבטחה
npm run test:security

# בדיקות ביצועים
npm run test:performance
```

### 3. בניית גרסה
```bash
# בניית צד לקוח
npm run build:client

# בניית צד שרת
npm run build:server

# בניית שירותים נוספים
npm run build:services

# בדיקת גרסה מקומית
npm run verify:build
```

### 4. פריסה לסביבת בדיקות
```bash
# פריסה לסביבת QA
npm run deploy:qa

# בדיקות אוטומטיות בQA
npm run test:qa

# בדיקות עומסים
npm run test:load
```
